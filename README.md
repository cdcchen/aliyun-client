# Aliyun Develop SDK

此SDK是在参考官方SDK的基础上开发的，思路基本跟官方的一致，增强了易用性。

阿里云SDK包括一系列的接口，目前统一在一个包下，后续接口越来越多的时候再考虑将包分拆开。目前在`namespace`上已经为将来分拆做了准备，所以分拆之后代码不需要修改。

#### [接口文档请查看Wiki](https://github.com/cdcchen/aliyun-client/wiki)
